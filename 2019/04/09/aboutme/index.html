通过 Hexo 搭建个人博客

Tue Apr 09 2019
  • 技术

一开始决定使用 Github Pages 的服务来搭建个人博客。官方推荐的是 Jekyll 来生成静态文件，但是看了下文档，不太适合我。我就转向了 Hugo。虽然 Hugo 速度很快，但是主题太少了，找不到自己喜欢的，最后投降了 Hexo。

# 流程

流程主要分为几步：搭建 -> 配置 -> 写作 -> 部署

根据文档来做没什么大问题。

部署

部署文档 (opens new window)

  • 安装：npm install hexo-deployer-git --save
  • 配置需要这样写：git@github.com:yourname/yourname.github.io.git
