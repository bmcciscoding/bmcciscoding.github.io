《深入理解计算机系统》是一本很经典的书，打算用一个季度精读，并总结笔记。 随着碰到的挑战越来越大，基础变得越发的总要。底层技术概念很难被时代取代，很多问题都答案在底层都设计上都做了很好都解答。 所以反复阅读，加深理解，这波绝壁不亏。

# 链接

理解链接过程，可以帮助你：

  1. 构造大型程序
  2. 避免一些危险的编程错误
  3. 理解语言的作用域规则是如何实现的
  4. 理解其他系统概念
  5. 理解利用共享库

iOS 程序都编译流程

clang -ccc-print-phases main.m   // 在终端执行该命令
+- 0: input, "main.m", objective-c
+- 1: preprocessor, {0}, objective-c-cpp-output
+- 2: compiler, {1}, ir
+- 3: backend, {2}, assembler
+- 4: assembler, {3}, object
+- 5: linker, {4}, image
6: bind-arch, "x86_64", {5}, image

源代码经过翻译之后编程.o文件，然后链接器把一大堆.o文件合成一个可执行程序：p。
