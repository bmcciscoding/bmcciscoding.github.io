# 关于 Dart

1、需要写分号，IDE 无法在自动添加分号 ：我感觉没啥用，字越多，阅读负担越重 2、关键字过多，需要熟能生巧，不熟多查阅文档 3、不理解为啥字符串是 String 大写，数字 num 是小写 4、..(级联调用) 看起来有点香

# 关于类

1、构造一个 const 的类很繁琐：属性必须都是 final，构造函数必须声明成 const

class Person {
  final num age; // 必须用 final
  const Person(this.age);  // 必须用 const
  const Person.custom(num age) : this(age * 10); // 必须用 const
  // 下面这个写法会报错
  const Person.fromData(Map<String, num> json) : age = json['x']
}

const zhangsan = Person(3) // 必须用 const

2、get set 方法丑陋

class Person {
  num age;
  num get fakeAge => age - 1;
  set fakeAge(num newValue) => age + 1;
}

# 关于 IDE

  1. 热重载太慢了：Reloaded 968 of 3617 libraries in 6,332ms.
  2. 切分支重新 build 时间太久了
  3. IDE 不顺手要磨合
  4. 编译太久了，出了问题很哪排查
  5. 经常找不到 devices
  6. 跑 debug 模式太卡了
  7. 闪退无法捕获堆栈信息
  8. 替换资源文件 hot reload 卡死
  9. VSCode 触发断点卡死，不懂是不是我设备的问题
