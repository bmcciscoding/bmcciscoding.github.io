# 工程结构

Xcode 里有 workspace、project、target、scheme 很过概念。可以简单理解为一个由大到小到变化。

# Project

Project 就是一个仓库，存放了所有都文件，资源，和构建需要都信息。一个 Project 可以包含很多个 Target，每个 Target 指定了如何去构建一个 products。而 Scheme 则是定义了个 Target 构建，配置，何时构建，何时测试的集合。
